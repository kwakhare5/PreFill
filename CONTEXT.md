# Instamart Intelligence Restock Chatbot Context

This document defines the canonical domain vocabulary for the Instamart Intelligence stateful multi-turn restock flow.

## Language

**Restock Alert**:
A notification sent to a household listing items predicted to deplete soon.
_Avoid_: stock warning, low-stock notification

**Cart**:
The active list of selected items and quantities ready for checkout.
_Avoid_: basket, shopping list, order items

**Checkout**:
The final step where a user confirms the cart to create and place an order.
_Avoid_: transaction, checkout completion, purchase
